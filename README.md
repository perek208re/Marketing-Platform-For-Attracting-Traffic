## Project Overview
**Watsons+** is a cross-platform marketing solution designed to bridge the gap between traditional email marketing and active social media engagement. Developed during a hackathon, the system creates a seamless lead-generation funnel that converts static email traffic into an interactive Telegram community through a reward-based incentive system.

## The Funnel Architecture
The platform operates through a multi-stage funnel designed to maximize user retention and data capture:
1.  **Email Outreach:** Automated marketing campaigns send high-conversion HTML templates to targeted users.
2.  **Incentivization:** Users receive a unique discount code or reward offer within the email.
3.  **Migration:** A "Get Discount" call-to-action redirects the user from their inbox directly to the **Watsons+ Telegram Bot**.
4.  **Conversion:** Upon entering the code in Telegram, the bot captures the user's ID and phone number, permanently linking them to the marketing database.

## Key Features
* **Advanced Admin Panel:** A WinForms-based dashboard allowing administrators to filter recipients by demographics (age, gender) and manage global mailing lists.
* **Automated Reward System:** Integrated verification logic that validates codes and triggers rewards for new subscribers.
* **Database Management:** SQL-backed infrastructure for real-time tracking of client interactions, Telegram IDs, and contact details.
* **Internationalization Support:** A smart detection system that adapts the language and tone of the message based on the user's country.
* **Admin Access Control:** Secure management of team permissions, allowing the addition and monitoring of administrative staff.

## Technical Stack
* **Backend & Logic:** C# (.NET)
* **Admin Interface:** WinForms (Windows Forms)
* **Database:** SQL (Client data storage & lead tracking)
* **Frontend (Email):** Custom HTML/CSS marketing templates
* **Bot Platform:** Telegram Bot API
